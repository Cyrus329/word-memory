# v39 方案2：直接导入，不依赖云端函数

这版主要解决：

1. 不再必须通过云端函数才能把手机更新到最新词库。
2. 新增 `专升本英语单词词库-v39一键导入-不走云端.json`。
3. 在网页里点“导入”，选择这个 JSON，会直接整体更新词库。
4. 导入时会保留已有学习进度、重点、熟悉度和学习时长。
5. 英语资料夹重新按当前词库生成，去掉了“第一章第三节 冠词与数词”残留。
6. 云同步仍保留，但改成轻量模式：只上传变化过的学习状态、自定义词和学习时长。

使用方式：

- 如果只想更新手机数据：打开手机网页 → 管理词库 → 导入 → 选择 `专升本英语单词词库-v39一键导入-不走云端.json`。
- 如果要连界面和资料夹一起更新：把本压缩包解压后，里面所有文件上传覆盖 GitHub 仓库根目录，再打开网页地址后面加 `?v=39` 刷新。

注意：

- 不要只上传 zip 文件本身，GitHub Pages 不会自动解压。
- 要把解压后的 `index.html`、`app.js`、`library-folder-data.js`、`service-worker.js` 等文件直接放到仓库根目录。
