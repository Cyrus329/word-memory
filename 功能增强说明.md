# 功能增强说明 v15

本版在原英语单词源程序上继续改，不是新建程序。

新增功能：

1. 三步背诵模式：英文 → 中文 → 例句/搭配，分步显示。
2. 中文选英文：根据中文释义做四选一。
3. 前缀 / 后缀 / 词根归类：按 re-、un-、-tion、port 等自动归类。
4. 形近词对比：按开头相近词分组，方便防止看错、拼错。
5. 同义词辨析：按“重要/帮助/改变/完成/影响”等意思归类。
6. 例句隐藏：英语资料夹里的中文、例句、备注默认收起。
7. 英译中 / 中译英：保留原程序训练模式。
8. 掌握等级：每个词可以标为 未学 / 认识 / 熟悉 / 掌握 / 稳定。
9. 发音：每个词保留美音、英音按钮。
10. 手机不卡：资料夹和智能归类都分批显示。

上传后建议用 `?v=15` 刷新。
