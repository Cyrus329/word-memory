# 专升本单词记忆 PWA 发布版

这个文件夹可以直接上传到 GitHub Pages。iPhone 用 Safari 打开 GitHub Pages 网址后，可以添加到主屏幕。

## 上传 GitHub

1. 登录 GitHub，新建一个仓库，例如 `word-memory`。
2. 把本文件夹里的所有文件上传到仓库根目录。
3. 打开仓库 `Settings` → `Pages`。
4. Source 选择 `Deploy from a branch`，Branch 选择 `main`，目录选择 `/root`，保存。
5. 等 GitHub Pages 生成网址，通常类似 `https://你的用户名.github.io/word-memory/`。

## iPhone 添加到主屏幕

1. 用 Safari 打开 GitHub Pages 网址。
2. 点底部分享按钮。
3. 选择“添加到主屏幕”。
4. 回到桌面，点“单词记忆”图标打开。

数据保存在手机浏览器本地。换手机前请先用软件里的“导出词库”备份。

## 2026-06-18 手机不卡版说明

这版不是新程序，是在原来的 word-memory-pwa 源程序上继续优化。

核心改动：
- 英语资料夹默认只显示文件夹和分组，不一次性展开全部单词。
- 点“打开本组单词”后只显示前 30 个，可继续“再显示 30 个”。
- 原词库列表默认只渲染前 40 个，可继续“再显示 40 个”。
- 搜索仍然从完整词库里搜索，不会漏词。

这样手机端不会因为一次性渲染几百个单词卡片而明显卡顿。
