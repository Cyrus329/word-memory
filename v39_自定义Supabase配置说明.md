# v39 自定义 Supabase 配置版

这版把 Supabase 项目地址和 anon public key 做进了云同步弹窗。

使用步骤：
1. 先在自己的 Supabase SQL Editor 运行 `supabase-setup.sql` 全部内容。
2. 上传本压缩包解压后的全部文件到 GitHub Pages 仓库根目录。
3. 打开网页，点云同步。
4. 填入自己的 Supabase Project URL 和 anon/public key。
5. 填公开编号、显示名称、编辑密码，再点“保存并开启同步”。

注意：网页地址只是分享链接用，不是 Supabase 地址。
