# v45 语音 / 云同步 / 保存修正说明

本版在 v44 基础上修正三个问题：

1. 语音播放卡顿
   - 读音不再优先请求在线 mp3，改为直接调用手机/浏览器系统英文语音。
   - 每次点读前会停止上一次读音，避免连续点击后排队，最后一次性卡爆。
   - 550ms 内重复点击同一个词会忽略，防止误触连点。

2. 云同步 JSON 报错
   - 修复 `Unexpected token 'u', "upstream..." is not valid JSON` 这类报错展示。
   - 现在会识别 Supabase 返回的非 JSON 错误，并显示中文原因。
   - 云端失败时不再反复自动同步，避免页面卡顿。

3. 保存逻辑
   - 本机学习记录仍保存到浏览器本地。
   - 云同步失败时会提示“本机已保存”，并自动关闭云同步，避免影响继续背词。

注意：如果要真正多人同步，需要 Supabase SQL/RPC 正常部署；如果只是自己背词，可以不打开云同步。
