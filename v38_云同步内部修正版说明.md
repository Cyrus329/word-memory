# v38 云同步内部修正版

这版修了两个内部问题：

1. 前端调用的云端函数名改为和 Supabase SQL 一致：
   - save_word_memory_cloud
   - load_word_memory_cloud
   - verify_word_memory_cloud_pin

2. 云同步不再把 1800+ 个词条逐条上传。
   现在只上传 1 个压缩云端包，里面包含：
   - 掌握等级
   - 重点词标记
   - 各模式复习进度
   - 自定义词
   - 总学习时长 / 今日学习时长

这样可以避免词库变大后保存超时。

上传后手机请用 ?v=38 打开。
